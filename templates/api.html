头像地址的生成

输入参数:

  • 服务端地址: server_addr
  • 用户名: user_name

    • 地址生成:

  • 1. 对用户名求md5值, 表示为16进制小写形式, 即finger_print = hex(md5(user_name))
  • 2. 将上一步所计算得到的finger_print值拼接在 http://server_addr/avatar/ 后构成头像地址

    • 代码示例:

        #!/usr/bin/python
    import hashlib

    server_addr = '210.25.189.114'
    user_name = 'zejiachen@gmail.com'
    finger_print = hashlib.md5(user_name).hexdigest()
    avatar_addr = "http://{0}/avatar/{1}".format(server_addr, finger_print)
    # 结果为: http://210.25.189.114/avatar/a69225e2777caebaa9a9db20d3bedeb5
    print avatar_addr
